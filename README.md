# D2TCDR
D2TCDR: Disentangled Diffusion-based Transfer for Cross-Domain Recommendation.

The implementation of the Diffusion Model refers to [DiffuRec](https://github.com/WHUIR/DiffuRec).

## Requirements

* python 3.8.19
* pytorch 2.4.1

## Usage

Data processing:
```bash
python process.py
```

Training and evaluation:
```bash
python main.py
```
